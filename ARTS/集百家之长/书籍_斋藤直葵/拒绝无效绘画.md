# 前言

- 何为画, 何为画得好?
	- 绘画: 将抽象情感化为有形之物的技巧, 用于瞬间传达某种印象
	- 画得好: 能够将这份情感传递给大家

# 画得不好

	- 比别人更热爱绘画!
