# 1 常规翻转

## 1.1 一个简单原理

```c++
while(需要的时候){
	先画一个"y";
	if(条件1)
		在 "y" 的交会点处画线[图1];
	else if(条件2)
		在 "y" 的交会点上方画线[图2];
	else
		在 "y" 的交会点下方画线[图3];
}
```
