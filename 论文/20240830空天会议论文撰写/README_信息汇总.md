			1. 若参加优秀报告专场，需投全文（见附件模版），原创性论文可评选 Oral 专场、竞争评奖之外，后续优先推荐到空天院中英文系列期刊
			2. 若不参加优秀报告专场，只投摘要（2-3 页，字数在 800 至 1000 字之间，包括论文标题、作者姓名和单位、关键词、研究背景、目的、方法、结果和结论等）
