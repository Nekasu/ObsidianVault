  • 2025-01-02：
       又在考虑买新电脑的事情，这次考虑的是 thinkbook 16+ 2024 32G+1T。因为国家可以补贴1449.8元。因此原本7299的价格现在仅有5799元了。再加上以旧换新的1400元，仅需要4399元就能拿下，好心动。
       但是在正式下单之前，需要思考一个问题，即我为何需要该电脑？最核心的理由只有一个：即我希望外接显卡的 oculink 不再占用我电脑的 m.2 硬盘位。新电脑自带 oculink 接口，且包含两个硬盘位，是我最心动的地方。为何需要两个硬盘位？因为我有许多游戏无法装入一个硬盘中。
       那么买个大硬盘不就解决了？我为何不买个大硬盘当系统盘呢？实际上，这个方法我也考虑过，没有执行下去的原因主要有两个：其一，无法舍弃硬盘中的数据；其二，2T的硬盘价格太高，需要约 1000 元。但现在我竟然考虑用"买个新电脑"的方法解决这个问题？真是太荒谬了。
       实际上还有一个更大的让我放弃买新电脑的因素：我希望且正在使用的是linux系统，我希望该系统能运行在我的笔记本上。在这种情况下，如果我换了新电脑，则也依旧会使用这块装载有linux系统的硬盘，而随新机附赠的硬盘可能会被我格式化。从结果上来看，这样也就是装了一块新的1T硬盘而已--毕竟我不刚需32G的内存。
       考虑到我想把电脑给越越姐打游戏这个问题，实际上一块大的硬盘也能解决这个问题。
    除了上面的硬盘问题，买新电脑就能获得两个好处：其一，新电脑自带 oculink 接口，是 PICE4.0 \(\times\) 4 的接口，而我改装的速度为 PCIE 3.0 \(\times\) 3，速度较慢；其二即为享受新电脑嘿嘿嘿嘿。以上三个好处是否值得我花费 5000 元？并不值得。5000 元不是一个小数字，有这个钱还不如再买个显卡扩展坞。
    因而可以有如下结论:
    • 我买新电脑的直接目标是节省下 m.2 硬盘位，核心目标是增加存储空间。
    • 之前没有买新硬盘是嫌硬盘贵（2T）且自己原本硬盘上数据重要。现在才换到linux系统，则没有这样的问题。
    • 直接使用国家补贴购买 2T 硬盘即可。
  • 2025-01-03：
       新硬盘已经到了，凯侠2T硬盘。现在要考虑这个硬盘是装windows系统还是linux系统（或者双系统）。需要分析一下装这两个系统分别的好处与坏处。
       装 linux 系统出于一个目的：Linux 系统是我新的玩具。linux 是我用的一个新的系统，所以我多体验体验这个对我来说几乎是全新的系统，探索未知总是令人着迷。
       装 windows 是为了游戏考虑。无论是外接显卡、2T硬盘、更换新电脑，都是为了同一个目的，那就是打游戏。由于我的 thinkbook14+ 2022 在性能上有所不足，所以改装了 oculink 接口，增加了外置显卡；由于使用了 m.2 接口改装为 oculink, 导致硬盘空间不足，所以在换新电脑还是购买新硬盘中纠结。
       这是二选一的选择题。在情感层面，我更希望使用 linux，这个系统好玩且有趣。从我的出发点看，我更应该在新硬盘中装 windows 系统。如果我装了 windows 系统，且添加了游戏，则可能会很少的将 linux 硬盘重新装入电脑。也即，我的 linux 之旅到此结束。
       这块新硬盘的最终宿命一定是为了游戏服务，因为我做这一切的出发点就是更好的玩游戏。而windows系统对游戏的支持更好，所以这块硬盘一定要装 windows（除非linux也能有如windows般的游戏支持，但实际上连原神、星铁、鸣潮的适配可能都不太好）。在这层意义下，这块硬盘一定不能装 linux，一定装 windows。
       解决了硬盘做什么用的问题后，还剩下两个问题需要想清：其一，这块硬盘要怎么装 windows ，迁移原本的 windows 系统盘，抑或新装一个 windows 系统？其二，现在电脑里的 linux 系统盘如何处置？让我们逐个思考。
       对于第一个问题，可以预测两种情况的结果。如果迁移了旧windows系统盘，那么就多了一块512G的空硬盘，这种情况下，我无法很好的利用硬盘资源，但会得到数据上的便利。如果安装新的系统盘，则一定会产生新的数据。这种不同数据在不同硬盘上的情况，实在是令人难以接受--每多一块硬盘就多一份数据，管理起来真的很麻烦。而且不管如何，装在1T硬盘上的游戏与软件数据都需要重新安装……权衡利弊之下，迁移硬盘数据是一个很好的选择。
       对于第二个问题，需要思考如何处理 linux 系统盘。由于我对 linux 系统的兴趣仍未消退，所以还是考虑先使用 linux 系统，而那块 2T 的硬盘，就先留着吧。
       哈哈，到最后发现，这块2T的硬盘实际上不需要现在立刻购买？
       不可否认的是，我在2025-01-02确实做出了"购买2T硬盘"的选择，并认为该选择是经过深思熟虑的结果，并立刻执行了购买2T硬盘的决定。但为何在2025-01-03看来，购买2T硬盘又不是一个必须的选择呢？反思看来，有几个原因。一，我在下定决心购买硬盘后，就没有再推演。实际上，在那时我应该考虑购买硬盘后的具体操作。也即，多看一步。
