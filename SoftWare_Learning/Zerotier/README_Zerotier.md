众所周知, 学校的服务器只能通过学校的内网连接. 如果在校外, 则无法连接, 那么有什么方法能解决呢？

可以使用 Zerotier 进行内网穿透, 为此, 我们需要一个 Zerotier 的账户充当管理员.
