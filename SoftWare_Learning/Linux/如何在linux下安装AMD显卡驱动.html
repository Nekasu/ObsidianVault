本教程来自于 该 bilibili 视频, 或可以阅读 AMD 官方安装教程以获取更详细的信息.
  1. 进入 AMD官网下载驱动。
  2. 更新 apt-get 包管理器 
                            sudo apt-get update 
                        sudo apt-get upgrade
  3. 安装 AMD 独显驱动的 安装包 
                            sudo apt-get install ./amdgpu-install-VERSION.deb
                        sudo apt-get update //检测更新
                        sudo apt upgrade //在线进行更新
  4. 使用 AMD 独显驱动安装包 安装必要的 包 
                            sudo amdgpu-install --usecase=rocm,hip,graphics --opencl=rocr //安装大部分图形驱动，已经可以打游戏跑AI了
                        sudo amdgpu-install --vulkan=amdvlk //安装vulkan
                    
  5.                         ls -l /dev/dri/render*
                        sudo usermod -a -G render "用户名"
                        sudo usermod -a -G video "用户名"
  6. 重启开始玩steam
