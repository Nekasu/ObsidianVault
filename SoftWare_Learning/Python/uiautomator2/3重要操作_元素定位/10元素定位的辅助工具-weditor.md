# weditor的作用的优势

1. 作用
	1. 可以辅助定位app上的某个元素
2. 优势
	1. 可以同时管理多个设备
	2. 可以直接在调试界面操作手机
	3. 可以自动生成uiautomator代码

# weditor的打开方式

在命令行中输入`weditor`即可在浏览器中打开当前设备
